ICML 2025
Blog Post | Paper

A benchmark that challenges AI agents on gravitational physics discovery tasks, evaluating scientific reasoning and observational planning capabilities within controlled, physics-based environments.

AstroCoder: AI-Powered Discovery and Usage of Astronomy Software

Application - April 10, 2025
Try AstroCoder | Technical Details

An AI-powered tool that helps astronomers discover and work with 2,270 repositories from the Astronomy Source Code Library (ASCL), providing AI-generated summaries, installation guides, and examples.

Calibrating and standardizing the Tip of the Red Giant Branch in the Small Magellanic Cloud using small-amplitude red giants
Nolan Koblischke, Richard I. Anderson
Astrophysical Journal, 2024
arXiv:2406.19375

Discovered that pulsation impacts the TRGB in the SMC as in the LMC, with older, longer period pulsators leading to a highly accurate TRGB tip measurement.

Astrophysical Journal Letters, 2024

Discovered that red giant pulsation adds a new dimension of the TRGB method, leading to an enhanced method for measuring the Universe's expansion rate.
